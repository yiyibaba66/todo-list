# 📝 Todo List 待办清单

一个简洁优雅的待办事项管理应用，帮助你高效管理日常任务。

## ✨ 功能介绍

- ✅ **添加待办**：输入任务内容，一键添加新待办
- ✏️ **标记完成**：点击复选框标记任务完成状态
- 🗑️ **删除任务**：移除不需要的待办事项
- 📊 **统计展示**：实时显示待办总数和已完成数量
- 🎨 **精美界面**：渐变背景、卡片式设计、流畅动画

## 🚀 运行方式

### 方式一：直接打开（推荐）
1. 下载或克隆项目
2. 双击 `index.html` 文件
3. 在浏览器中即可使用

### 方式二：本地服务器（开发模式）
```bash
# 安装 http-server（可选）
npm install -g http-server

# 启动服务器
http-server -p 8080

# 打开浏览器访问
http://localhost:8080
```

## 🌟 项目特点

- **响应式设计**：适配不同屏幕尺寸
- **零依赖**：无需安装额外依赖，开箱即用
- **数据持久化**：刷新页面后数据不丢失（可选扩展）
- **代码规范**：结构清晰，易于理解和维护
